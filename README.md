# 周报小工具 WeekRepo

> 一款免费的 Windows 桌面周报助手：按真实工作日提醒你记录工作，一键生成、AI 优化并归档你的周报。
>
> 本仓库仅用于**发布版本**（[Releases](https://github.com/KatouRyuuji/WeekRepo/releases)），项目暂不开源。

## 下载安装

1. 前往 [Releases 页面](https://github.com/KatouRyuuji/WeekRepo/releases/latest)，下载最新的 `WeekRepo-vX.Y.Z.exe`。
2. 双击运行即可，**无需安装、无需 Python 环境**（单文件绿色版）。
3. 首次运行时 Windows SmartScreen 可能提示"未知发布者"，点击「更多信息 → 仍要运行」即可。

> 系统要求：Windows 10 / 11。

## 功能一览

- **跟随真实工作日**：按中国法定节假日与调休安排判断工作日，节假日绝不打扰。
- **智能定时提醒**：每周首个工作日提醒填写周计划，工作日晚间提醒记录日报，最后一个工作日提醒确认本周总结；错过提醒会延迟再提醒。
- **一键格式化导出**：自动生成规范的 Markdown 周报，连同配图按周归档到「周报 (起始日期 ~ 结束日期)」文件夹。
- **一键 AI 优化**：内置精调提示词，自动分类归纳、提升专业度，生成适合直接汇报的版本。
- **一键直达报告**：工作区可一键打开本周周报（未导出会自动导出）、打开 AI 汇报版报告（未优化会自动优化）、打开本周周报文件夹。
- **周报配图**：为每周上传配图，与周报统一归档。
- **深色 / 明亮主题**、系统托盘常驻（双击托盘图标唤回窗口）。
- **单实例运行**：重复启动会自动唤出已打开的窗口，不会开出第二份。
- **自动更新**：每次启动自动检查本仓库的最新版本，一键完成更新。

## AI 优化怎么启用？

两种方式任选其一（在应用「设置」中操作）：

1. **激活码兑换**：输入作者分享的 10 位激活码，一键启用内置 Kimi 模型，开箱即用。激活码可通过作者的 B 站主页（space.bilibili.com/445111）获取。
2. **自备 API**：在「设置 → AI 配置」填写任意兼容 Anthropic Messages 协议的服务地址、Token 与模型名。

## 自动更新说明

应用每次启动会静默检查本仓库的最新 Release：

- 检测到新版本时会弹窗询问是否自动更新，确认后自动下载、校验并重启完成替换；
- 如你的网络访问 GitHub 不佳，可随时手动到 [Releases](https://github.com/KatouRyuuji/WeekRepo/releases/latest) 下载最新版覆盖旧文件即可；
- 你的所有数据与设置都保存在本机用户目录（见下），**更新/覆盖 exe 不会丢失任何数据**。

## 数据保存在哪里？

- 记录与设置：`C:\Users\<用户名>\AppData\Local\WeeklyReportTool\data`
- 导出的周报与配图：`文档\WeeklyReports`（可在「设置」中修改）

## 常见问题

**Q：关闭窗口后程序去哪了？**
A：默认最小化到系统托盘继续按时提醒，双击托盘图标即可唤回；关闭时也可以选择「直接退出」。

**Q：提醒没有弹出？**
A：请确认程序在运行（系统托盘有图标）。程序会自动联网同步最新节假日安排，离线时使用内置数据。

**Q：杀毒软件 / SmartScreen 报警？**
A：单文件 PyInstaller 程序的常见误报。程序不含任何恶意行为，介意可在 Releases 页核对文件大小与 sha256 摘要后放行。

**Q：如何反馈问题或建议？**
A：欢迎在本仓库提 [Issue](https://github.com/KatouRyuuji/WeekRepo/issues)，或到 B 站私信作者。
